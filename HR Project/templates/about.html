About the System

NextGen HR is an AI-powered Resume Screening System fully engineered to optimize and streamline candidate pipelines. Powered by state-of-the-art Natural Language Processing.

Core Functionality

The system instantly parses unstructured resumes, automatically extracting crucial data like candidate names, core skills, and exact years of professional experience across PDF and DOCX formats.

By actively cross-referencing this extracted timeline against dynamic HR-defined job requirements, the AI model generates real-time, bias-free match scores to isolate top talent.

Technology Stack

Backend Architecture: Python 3 / Flask
AI Models: NLP / Random Forest
Document Engines: PyMuPDF / docx
Frontend UI: HTML5 / CSS Glassmorphism
